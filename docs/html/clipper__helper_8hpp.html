Detailed Description

Clipper Library Bridge/Helper.

This header contains functions used to inflate/deflate Polygon objects from the AppliedRoboticsEnvironment(*). Functions act as a bridge for the Clipper library(**) that uses its own types and classes.

(*) https://github.com/ValerioMa/AppliedRoboticsEnvironment/blob/master/src/9_project_interface/include/utils.hpp (**) http://www.angusj.com/delphi/clipper.php
