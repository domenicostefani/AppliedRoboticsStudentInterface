std::vector< Position > dubins::Curve::discretizeSingleCurve ( double  delta)

Discretize a SINGLE Dubins Curve.

Discretize a single Dubins curve, sampling positions from the curve with fixed distance delta. The fact that it's a single curve means that there is no need to carry over the delta remainder value or the last abscissa. NOTE: Using this for a multicurve/multipoint causes inconsistency of sample spacing at the end of each curve.

Parameters
[in]deltasampling step size
Returns
vector of path samples (positions)
